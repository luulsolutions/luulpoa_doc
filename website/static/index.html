Beautyful Features

The goal of LuulPOS is to make it a full-blown Point-Of-Sale, which complete well-designed POS features including support for:-

  • Quick keys and/or product setup/lookup
  • Client/shop setup
  • User accounts and permissions
  • Integration with popular/custom Payment Providers
  • Multiple payment methods/split payments
  • Returns, refunds and store credit features
  • Mobile Registers such as iPad, Android Tables, small screen phones
  • Customer-facing display and Admin back-office
  • Inventory management
  • Bulk product imports
  • Product variants and composites
  • Barcodes
  • Stock and auto-filled orders
  • Inventory counts
  • Stock Transfers
  • Promotions
  • Customer Management
  • Employee Timesheet
  • Loyalty program
  • Dashboards

Real Time / Offline Systems

To help boost performance and reduce load to backend servers, LuulPOS syncs clients apps with servers. We have plans to support complete offline client operations which allows clients to function even with no Internet/server connection. At later time client apps upload transactional data to servers.
